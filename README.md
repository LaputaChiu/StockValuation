## About The Project

A basic stock valuation tool with *Yahoo Finance*.
The tool is for testing purpose only, please **DONT** use it as investment guide.

## Getting Start

### Prerequisites

Install *yahoo finance*
```cmd
pip install -Iv yahoo-fin==0.8.9.1
```
